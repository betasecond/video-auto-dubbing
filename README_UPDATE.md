### 4. 新增故障排查章节

添加了常见问题的详细解决方案：

#### 问题 1: 任务卡在"等待处理"
```bash
# 检查 worker 状态
docker-compose exec worker celery -A app.workers.celery_app inspect active
```

#### 问题 2: Redis 连接失败
- 确保无 `REDIS_PASSWORD` 配置
- 检查 Docker Compose Redis 无密码参数

#### 问题 3: 数据库连接失败
```bash
# 检查环境变量
docker-compose exec api env | grep DB_
```
